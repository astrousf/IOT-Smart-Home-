// Core Systems

Everything Your Home Needs,
Nothing It Doesn't

Six tightly integrated subsystems working in concert — from solar-powered sensors to live camera feeds — all orchestrated from a single dashboard.

🧠

Distributed Microcontroller Architecture

A master-slave network of one ESP32 and two NodeMCUs divides work by physical zone. If one node drops offline, every other room stays fully operational. Wireless node communication over Wi-Fi eliminates long-distance cabling entirely.

ESP32 · NodeMCU · Wi-Fi Mesh
☀️

Green Energy & Smart Power

Solar panel charges an 18650 Li-Ion cell via TP4056. A custom voltage divider reads real-time battery level as a 0–100% indicator — just like your phone.

Solar · TP4056 · Telemetry
🛡️

"Fort Knox" Security

RFID keyless garage entry, ESP32-CAM live streaming to dashboard, and PIR motion sensors covering every room for intrusion detection and smart lighting automation.

RFID · ESP32-CAM · PIR
🚨

Automated Safety Protocols

MQ-2 gas sensor and flame detector continuously monitor the kitchen. On hazard detection, the system triggers exhaust fans automatically — no human intervention needed.

MQ-2 · Flame · Auto-Mitigation
🌬️

Climate & Actuation Control

DHT11/22 sensors track temperature and humidity live. L298N motor drivers handle garage doors and smart curtains; TIP122 transistors control DC fans safely.

DHT · L298N · TIP122
💻

Dashboard

A RESTful Laravel backend handles all microcontroller POST/GET requests. The sleek, responsive frontend lets you monitor sensors, view live cameras, and override any actuator from any device.

Laravel · REST API · Responsive
// System Flow

How It All Works Together

01

Sensors Collect Data

DHT, PIR, MQ-2, and voltage sensors continuously sample the environment across all zones.

02

Nodes Process & Relay

Each microcontroller handles its zone and relays data to the master ESP32 over Wi-Fi.

03

Backend Stores & Serves

The Laravel API receives POST requests, stores telemetry, and exposes it via clean endpoints.

04

You Stay in Control

The web dashboard displays everything in real time. Override any actuator with a single tap.
